Pour compiler :

- Se placer dans le dossier SDL et écrire en ligne de commmande :
    <br>`gcc src/*.c -o bin/prog -I include -L lib -lmingw32 -lSDL2main -lSDL2`  

Pour exécuter :

- Se placer dans le dossier SDL et écrire en ligne de commmande :
    <br>`bin\prog.exe`
