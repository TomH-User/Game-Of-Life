Implémentation du Jeu de la Vie en C

![](Images/Image.jpg)

Pour compiler :

- Se placer dans le dossier SDL et écrire en ligne de commmande :
    gcc src/*.c -o bin/prog -I include -L lib -lmingw32 -lSDL2main -lSDL2  

Pour exécuter :

- Se placer dans le dossier SDL et écrire en ligne de commmande :
    bin\prog.exe
